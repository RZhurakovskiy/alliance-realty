Политика обработки персональных данных

Настоящая политика обработки персональных данных определяет порядок обработки и защиты информации, предоставляемой пользователями сайта агентства недвижимости Alliance Realty.

1. Общие положения

1.1. Настоящая политика разработана в соответствии с Федеральным законом РФ «О персональных данных».

1.2. Администрация сайта alliance-realty.ru обязуется обеспечить защиту ваших персональных данных.

2. Состав обрабатываемых персональных данных

2.1. Персональные данные, которые вы можете предоставить: имя, контактный телефон, адрес электронной почты. Эти данные используются для предоставления услуг, обратной связи и отправки актуальных предложений.

3. Цели обработки персональных данных

  • Предоставление услуг агентства недвижимости;
  • Консультации по объектам недвижимости;
  • Информирование об акциях, новостях и новых объектах.

4. Защита данных

4.1. Мы применяем современные технологии шифрования и защиты данных для предотвращения их утечки и несанкционированного доступа.

4.2. Доступ к персональным данным имеют только уполномоченные сотрудники.

5. Контакты

Если у вас есть вопросы по поводу обработки ваших персональных данных, свяжитесь с нами по телефону: +7 999 999-99-99 или через электронную почту: info@alliance-realty.ru.
